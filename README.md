# MATLAB-Workplace-Challenge-E.P.P-Team-2---Data-Analytics-Apps-for-a-Smart-Eco-Gym
Project to complete in the 2026 MATLAB Workplace Challenge as a part of the Engineering Pathway Program. Team of four tasked with creating a data analytics app/dashboard from data collected in smart eco gyms utilizing MATLAB / Simulink. 

Commit (7/21/2026): Data organized now into tables. fullSessionData is the combination of all three data given. Datasets people and activity_met_referenced joined onto session table. Now each session has specific user data and the met for the activity given. Also calculated calories burned for each session using the formula given in project instructions GitHub ("calories = MET * weight_kg * duration_hours"). Gives a graph on a person's calories burned per session vs personal average calories burned.
